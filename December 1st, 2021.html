      • {{[[TODO]]}} more on the different tagging using different references:
        • i now think the main difference is whether:
          • a) you want your page to be a "blog", i.e. linked mentions are in the historical order (oldest first, newest last), thus reading top-to-bottom, thus encouraging you to write not inside the page, but inside the daily notes, under that pages' tag.
            • good examples of this are: [[git-rebase]]; [[i just re-discovered conway's law (public)]] (if refactored into daily notes); etc
          • b) you want to use the page for tagging, i.e. to get latest updates, thus linked mentions should be as they are in #[[roam]] - bottom-to-top (newest first, oldest last)
            • a good example of this is e.g. [[codemods]], [[roam-traverse-graph]] itself
          • i think B is the preferred default, because that's how it is in #[[roam]], and then e.g. adding a tag `#make-blogpost-out-of-linked-mentions` (or whatever) would be a good way to swap the ordering of the linked mentions. though, if the default stays the same as in roam, it kinda sucks because a person would be less likely to discover it.
